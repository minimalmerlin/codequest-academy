# CodeQuest Academy – Lernkonzept (für Eltern)

## Ziel
Deine Kinder lernen Programmieren **spielerisch** – von den ersten Schritten bis zu ersten fertigen Projekten, die man veröffentlichen kann.

**Sprachen & Fokus**
- **HTML/CSS**: Webseiten bauen (Struktur, Design, Inhalte)
- **JavaScript**: Logik & Interaktion (Entscheidungen, Schleifen, Funktionen, kleine Games)
- **Python**: Logik & Story/Tools (lesbar, vielseitig, später auch Daten/AI)

## Didaktik (so funktioniert das Lernen)
- **Quests**: 10–30 Minuten, 1 klarer Skill pro Einheit
- **Sofort‑Feedback**: Code ausführen + Check (Output/Struktur)
- **Gamification**: XP, Level, Streak; Fortschritt lokal gespeichert
- **Spiralprinzip**: Konzepte kommen wieder – jedes Mal etwas schwerer
- **Projekte**: Nach jedem Block ein „sichtbares Ergebnis“

## 8‑Wochen‑Plan (empfohlen)
Rhythmus: **3× pro Woche 20–30 Minuten**

1. **Woche 1–2 (Web)**: HTML‑Basics, Links/Bilder, Listen, erstes CSS, Steckbrief‑Projekt
2. **Woche 3–5 (JavaScript)**: Variablen, if/else, Schleifen, Funktionen, kleines Quiz
3. **Woche 6–7 (Python)**: print/Variablen, if/else, Schleifen, Listen, Text‑Adventure
4. **Woche 8 (Capstone)**: Mini‑Spiel im Browser + Veröffentlichung (Git/Vercel)

## Projekte (Beispiele)
- **Meine persönliche Homepage** (Web): Steckbrief + Styling
- **Quiz‑Spiel** (JS): Punkte zählen, Auswertung, später Buttons/UI
- **Text‑Adventure** (Python): Story mit Entscheidungen
- **Capstone** (Web+JS): Mini‑Spiel im Browser

## Eltern‑Rolle (kurz & effektiv)
- Lass dein Kind **erklären**, was der Code macht (2 Minuten reichen).
- Stelle 1–2 Fragen: „Was passiert, wenn…?“
- Feiere Bugs: „Nice – wir haben was entdeckt!“

## Technische Hinweise
- JavaScript läuft im Browser in einer **Sandbox (iframe)**.
- Python läuft im Browser über **Pyodide** (beim ersten Start wird es aus dem Internet geladen).
